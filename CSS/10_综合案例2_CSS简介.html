CSS (层叠样式表)

层叠样式表(Cascading Style Sheets, 缩写为CSS), 是一种样式表语言, 用来描述HTML或XML(包括如SVG、MathML、XHTML之类的XML分支语言)文档的呈现。 CSS描述了在屏幕、纸质、音频等其它媒体上的元素应该如何被渲染的问题。

CSS是最开放的网络核心语言之一，由W3C规范实现跨浏览器的标准化。 CSS节省了大量的工作。样式可以通过外部定义的.css文件中, 同时控制多个网页的布局，这意味着开发者不必经历在所有网页上编辑布局的玛法。 CSS被分为不同等级；CSS1现已一起，CSS2.1是推荐标准， CSS3分成多个小模块且正在标准化中。
